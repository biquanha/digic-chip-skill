# 数字芯片开发标准库

这是一个面向数字芯片研发场景的标准化技能库与流程库。

当前已沉淀章节：
- `skills/rtl-remote-vcs-wave`：在远端 VCS 节点执行 RTL 编译、仿真、FSDB 导出与 `fsdbreport` 波形摘要分析。

建议组织方式：
- `skills/`：面向 agent 的可复用技能
- `commands/`：高频命令模板
- `docs/`：使用说明与环境说明
- `scripts/`：仓库级辅助脚本
