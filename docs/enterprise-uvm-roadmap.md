# 企业级 UVM 验证平台路线图

本文给出从“新模块第一版平台”到“企业级可维护平台”的推荐演进路线。

## 阶段 1：最小闭环
- 远端环境打通
- smoke case 跑通
- FSDB 生成
- scoreboard 有最小比较

## 阶段 2：平台成形
- testcase 分层
- coverage 起步
- reference model 稳定
- RAL 最小接入

## 阶段 3：企业级增强
- 回归分层
- 断言体系
- lint / CDC / formal 接口
- 工具链版本管理

## 阶段 4：交接与维护
- 平台说明完整
- 维护规范明确
- 验收口径固定
- 新成员可按文档接手
