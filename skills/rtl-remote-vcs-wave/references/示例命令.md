# 示例命令

## 远端一键执行

```bash
bash scripts/run_remote_vcs_fsdb.sh \
  --remote eda01 \
  --workdir /nfs/home/USER/project/rtl_min_case \
  --module-vcs synopsys/vcs/Q-2020.03-SP2 \
  --module-verdi synopsys/verdi/R-2020.12-SP1 \
  --module-license license \
  --verdi-home /nfs/tools/synopsys/verdi/R-2020.12-SP1
```

## 预期产物

- `simv`
- `simv.log`
- `counter.fsdb`
- `fsdb_summary.txt`
- `fsdb_strobe.txt`

## fsdbreport 示例

```bash
fsdbreport counter.fsdb -s /tb_counter/count -o fsdb_summary.txt
fsdbreport counter.fsdb -strobe "/tb_counter/clk==1" -s /tb_counter/en -s /tb_counter/count -o fsdb_strobe.txt
```
